# 월별 성과 추이 대시보드 PRD

## 1. 배경

### 현재 상태

대시보드의 핵심 지표들이 **전체 기간 누적 데이터**로만 산출되고 있다.

| 섹션 | 현재 제공 방식 | 한계 |
|------|---------------|------|
| 섹션 1: 포트폴리오 요약 | 전체 기간 합산 (총 수익률, 승률 등) | 최근 성과가 좋아지는지 나빠지는지 알 수 없음 |
| 섹션 2: 월별 성과 | 매수/매도 건수·금액만 표시 | 승률, 손익비 등 핵심 지표 없음 |
| 섹션 4: 투자 지표 | 전체 기간 평균 수익률·손실률·손익비 | 월별 변화 추이 없음 |
| 섹션 5: 매매 인사이트 | 기대값·Profit Factor 전체 기간 합산 | 시간에 따른 개선/악화 파악 불가 |

**예시 문제**: 전체 승률이 55%라고 하면, 초기에 40%였다가 최근 70%로 개선된 것인지, 반대로 70%에서 40%로 하락한 것인지 구분할 수 없다.

### 기대 효과

월별로 핵심 성과 지표를 추적하면:
- **성장 확인**: 투자 실력이 개선되고 있는지 객관적으로 판단
- **문제 조기 발견**: 특정 지표가 악화되는 추세를 조기에 감지
- **전략 평가**: 매매 전략 변경 전후의 성과를 비교
- **규율 모니터링**: 손절/익절 규율이 일관되게 유지되는지 확인

## 2. 요구사항

### 2.1 새 섹션: 월별 성과 추이 (섹션 6)

기존 섹션 5 하단에 **섹션 6: 월별 성과 추이**를 추가한다. 각 행이 하나의 월(YYYY-MM)을 나타내는 테이블 형식이다.

#### 월별 추이 테이블 컬럼

| 컬럼 | 지표 | 산출 방식 | 현재 제공 여부 |
|------|------|----------|--------------|
| A | 연월 | YYYY-MM | 섹션 2에 있음 |
| B | 매도건수 | 해당 월 매도 거래 수 | 섹션 2에 있음 |
| C | 실현손익(원) | 해당 월 매도 profit_krw 합계 | 섹션 2에 있음 |
| D | 수익률(%) | 해당 월 실현손익 / 매도금액 | 섹션 2에 있음 |
| E | **승률(%)** | 해당 월 수익 매도 건수 / 전체 매도 건수 | **신규** |
| F | **평균 수익률(%)** | 해당 월 수익 매도의 profit_rate 평균 | **신규** |
| G | **평균 손실률(%)** | 해당 월 손실 매도의 profit_rate 평균 | **신규** |
| H | **손익비** | \|평균 수익률\| / \|평균 손실률\| | **신규** |
| I | **Profit Factor** | 총 수익금액 / \|총 손실금액\| | **신규** |
| J | **기대값(원)** | (평균수익 × 승률) - (평균손실 × 패률) | **신규** |
| K | **전월대비 손익 변화(%)** | (당월손익 - 전월손익) / \|전월손익\| × 100 | **신규** |

> **그룹핑 기준**: 연월(YYYY-MM) 단독. 계좌 구분 없이 전체 합산으로 추이를 본다.
> (계좌별 분리가 필요하면 추후 확장)

#### 데이터 정렬
- 연월 오름차순 (가장 오래된 달 → 최신 달)

### 2.2 기존 섹션 2 (월별 성과) 처리

섹션 2는 (연월, 계좌) 기준 계좌별 상세로 유지하고, **섹션 6을 신규로 추가**하여 전체 합산 월별 추이를 제공한다.

- 섹션 2: 계좌별 매수/매도 건수·금액 상세 (기존 유지)
- 섹션 6: 전체 포트폴리오 합산 핵심 성과 지표 추이 (신규)
